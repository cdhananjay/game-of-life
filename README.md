# game of life

# [cdhananjay.github.io/game-of-life](https://cdhananjay.github.io/game-of-life/)

A cellular automaton simulation by John Conway where you create patterns that evolve over time.

Click cells to bring them to life, then watch them grow, change, or fade away.

## The Rules

- cell dies if it has less than 2 or more than 3 neighbours
- dead cell becomes alive if it has exactly 3 neighbours
